
************************************MOBILE AUTOMATION FOR STATEFARM**************************************

Overview - This framework is developed to automate web application on mobile browser. Here we have separated the data and objects in separate file, so it would be easy to maintain if any changes required in objects or test data later on.
We have four classes in this framework listed below along with their objective

ControlFile - It is used to handle all the operations related to Browser and Mobile emulation
PageObjects - It is used to store the information to identify the webelements on page using any locator
TestData - It is a property file and used to store data in Key Value pair
TestScript1 - This is test script file where we are using all reusuable components to run script on mobile browser

Technology Used - Java, Selenium, Maven, TestNG

PreRequisite - Chrome version 85 should be installed on machine
